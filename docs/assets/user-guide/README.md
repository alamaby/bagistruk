# User Guide Placeholder Assets

Folder ini berisi placeholder SVG untuk `docs/user-guide.md`.

Ganti placeholder dengan screenshot aplikasi asli saat UI siap dipublikasikan. Screenshot yang disarankan:

- `scan-flow.svg`: tab Scan dengan preview foto, tombol Add Photos, tombol Scan, dan status processing/credit jika relevan.
- `review-bill.svg`: layar Review bill setelah OCR sukses, menampilkan merchant, item, pajak/service, total, dan tombol Simpan Bill.
- `split-bill.svg`: layar Split bill dengan beberapa partisipan, avatar aktif, item yang sudah di-assign, dan indikator item belum/sudah dibagi.
- `settlement.svg`: layar Detail Tagihan dengan status lunas/belum lunas, daftar partisipan, tombol share, switch pembayaran, dan export Plus.
- `history.svg`: tab History dengan summary card, banner window History, daftar bill, dan monthly insight untuk akun Plus.
- `settings.svg`: tab Settings akun terdaftar dengan credit scan, billing, preferensi, info bank transfer, dan bill terhapus.
- `plan-comparison.svg`: layar billing/paywall atau komposit screenshot yang menjelaskan Anonymous, Free, dan Plus.
