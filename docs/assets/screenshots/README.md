# Screenshot Placeholders

Replace these SVG placeholders with final screenshots before publishing the repository homepage or Play Store listing.

Suggested final captures:

- `scan-receipt`: receipt capture and multi-photo picker flow.
- `review-bill`: OCR result review and editable line items.
- `split-bill`: participant assignment flow.
- `settlement`: bill detail and payment status tracking.

Recommended Play Store screenshot format: PNG or JPEG, portrait phone screenshots, consistent device frame or no frame across all images.
