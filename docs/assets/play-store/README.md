# Play Store Asset Placeholders

Store source assets for the Play Store listing here.

Recommended assets:

- Feature graphic: `feature-graphic.png`, 1024x500.
- App icon source: 1024x1024 PNG with adaptive-icon safe padding.
- Phone screenshots: scan, review, split, settlement.
- Tablet screenshots if tablet support is part of the release target.

Keep final exported assets out of local-only folders so release reviewers can inspect them from the repository.
